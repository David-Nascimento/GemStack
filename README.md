## 🛠️ Steps
## Verifica a versão do Ruby
```sh
Ruby -v
```
## API e WEB
Se quiser executar os testes WEB 
```sh
cd frontend
bundle install
cucumber
```
Se quiser executar os teste API
```sh
cd backend
bundle install
rspec
```

Para Executar os containers
```sh
cd infra
```
